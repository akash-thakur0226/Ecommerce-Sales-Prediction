# 🛒 E-commerce Sales Prediction

## 📌 Problem Statement

Predict sales (TotalPrice) using historical transaction data.

## 📊 Dataset

Online Retail Dataset (UCI Repository)

## 🧹 Data Cleaning

* Removed missing CustomerID
* Removed negative values
* Converted date column

## 🧠 Feature Engineering

* Created TotalPrice
* Extracted Month, Day, Hour

## 📈 EDA

* Analyzed monthly sales trends
* Identified top countries and products
* Studied hourly sales patterns

## 🤖 Models

* Linear Regression
* Random Forest

## 📏 Evaluation

* MAE
* MSE

## 🔥 Key Insights

* Sales peak during holiday season
* UK dominates revenue
* Random Forest performs better

## 🚀 Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
